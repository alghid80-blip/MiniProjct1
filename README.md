# MiniProjct1
Cleaning Dataset – missing values & Outliers

الخطوة الأولى: حملت الملف وطلعت أول خمس صفوف وشفت أنواع الأعمدة وعدد القيم المختلفة

الخطوة الثانية: حددت نوع العمود وش نسوي فيه لو فيه missing values عشان يعرف كيف يتعامل مع البيانات اللي مو مرتبة

الخطوة الثالثة: حولنا الأعمدة الرقمية لـ float والتاريخ لـ datetime وأي حاجة غلط أو NaN يتحول لـ datetime

الخطوة الرابعة: عوضنا القيم المفقودة missing values بالـ median لكل عمود رقمي

الخطوة الخامسة: كيف نتعامل مع Outliers أي قيمة كبيرة مرة نقصها عشان تكون مناسبة مع باقي القيم بالداتا

الخطوة السادسة: نظفنا النصوص ومسحنا الفراغات وخليت كل الكلام lower عشان يكون كل شيء موحد
